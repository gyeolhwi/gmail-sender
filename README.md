# 거래명세서 자동 발송 (Google Apps Script)

> 구글 시트에 거래처·청구 내용만 적어두면 **거래명세서 PDF를 자동 생성해 거래처 이메일로 발송**하고 Google Drive에 보관합니다. 서버·비용 없이 구글 계정 하나로 동작합니다.

- **데이터** Google Sheets · **출력** PDF (한국식 거래명세서) · **발송** Gmail · **보관** Google Drive
- **자동화** 매월 지정 날짜·시각 자동 발송 (또는 버튼으로 수동)
- **비용** 무료 (개인 Gmail 하루 100통 한도 내)

---

## ✅ 셋업에서 사람 vs 자동 (핵심)

> 목표: **사람이 꼭 해야 하는 것만 남기고 나머지는 전부 자동.**

### 👤 사람이 꼭 해야 하는 것 (이것뿐)
1. **구글 시트 생성 + 양식 가져오기** — `docs/거래명세서양식.xlsx` 를 시트로 가져오기 (1분)
2. **권한 승인** — 스크립트 첫 실행 때 OAuth 동의 (보안상 사람이 클릭)
3. **내 데이터 입력** — `설정`(회사정보)·`거래처`(이메일)·`정기항목`(매달 청구) 채우기
4. **실제 발송 ON** — 검증 끝나면 `발송_활성화 = TRUE`

### 🤖 자동으로 되는 것 (코드/메뉴가 처리 — 사람 손 안 감)
- 시트 6개 생성, 양식 한 시트로 통합, **예시 데이터·드롭다운·입력 형식 검증·금액 수식** 전부 자동
- 정기항목 → 거래내역 **자동 채움**, PDF 생성·Drive 보관·발송로그 기록
- 매월 자동 발송(트리거), 발송 전 미리보기, 중복/보류 처리, 결과 요약 메일

> AI 에이전트로 코드 배포까지 자동화하려면 → **[AGENTS.md](AGENTS.md)** 참고 (clasp 기반).

## 🚀 빠른 시작 (사람용, 복붙)

1. **양식 가져오기**: 새 구글시트 → `파일 → 가져오기` → `docs/거래명세서양식.xlsx` → **"새 시트 삽입"**
   → `자료입력`·`세금계산서양식` 탭 생성
2. **코드 붙여넣기**: `확장 프로그램 → Apps Script` →
   - `프로젝트 설정`에서 "appsscript.json 표시" 체크 → `gas/appsscript.json` 붙여넣기
   - `Code.gs` ← `gas/Code.gs`, `Setup.gs`(파일 추가) ← `gas/Setup.gs` → 저장
3. **설치**: 시트 새로고침(F5) → 메뉴 `📨 거래명세서 → ⚙ 관리 → 설치 / 갱신` → 권한 승인(`고급 → 이동 → 허용`)
4. **내 데이터 입력** + **테스트** → 문제없으면 `설정`의 `발송_활성화 = TRUE`

> 자세한 단계·문제해결은 **[docs/SETUP.md](docs/SETUP.md)**.

## ✏️ 어디에 무엇을 넣나 (사람 입력)

| 시트 | 넣을 것 |
|------|---------|
| **설정** | 회사명·사업자번호·대표자·주소·입금계좌 / 발송일·발송시각 / 관리자·테스트 메일 |
| **거래처** | 거래처ID(C001…)·상호·이메일 (필수), 사업자번호 등 선택 |
| **정기항목** | 매달 청구할 품목·수량·단가 (한 번만 적어두면 매달 자동) |
| 세금계산서양식의 은행/계좌 칸 | PDF에 찍히는 계좌 (양식에 직접) |

> 코드의 모든 값은 **예시(placeholder)** 입니다. 실제 데이터는 **본인 구글 시트에만** 들어갑니다.

## 🗓 매월 사용

1. `정기항목` 관리(대부분 1회 등록 후 그대로)
2. `📨 거래명세서 → 👀 미리보기` — 발송 없이 PDF만 확인
3. `📧 이번 달 발송하기` — 확인창에서 건수 확인 후 발송
4. 또는 `⚙ 관리 → 자동발송 켜기` — 매월 자동

**발송 대상** = 이번 달(당월) + `미발송` + 거래처 `발송여부=TRUE` 인 건만 (`발송_활성화=TRUE`일 때).
특정 행 제외 → `거래내역`의 `발송상태`를 **`보류`** 로.

## ⚠️ 알아둘 점
- 개인 Gmail 하루 **100통** 한도 (Workspace 1,500통). 소량이면 무관.
- `발송_활성화 = FALSE` 면 PDF만 만들고 메일은 안 감(안전 테스트용).
- 자동발송은 구글 트리거 특성상 지정 시각의 **1시간 이내** 실행. 즉시 검증은 `⚙ 관리 → 테스트: 1분 후 자동발송`.
- 사업자번호 칸은 사업자번호/단체 고유번호(10자리)용. 하이픈 있게/없게 입력해도 자동 정렬.
