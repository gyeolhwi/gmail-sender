# 거래명세서 자동 발송 (Google Apps Script)

> Google Sheets에 거래처·청구 내용을 적어두면, **거래명세서 PDF를 자동으로 만들어 거래처 이메일로 발송**하고 Google Drive에 보관합니다. 서버·비용 없이 구글 계정 하나로 동작합니다.

- **데이터**: Google Sheets · **출력**: PDF (한국식 거래명세서 양식) · **발송**: Gmail · **보관**: Google Drive
- **자동화**: 매월 지정한 날짜·시각에 자동 발송 (또는 버튼으로 수동 발송)
- **비용**: 무료 (개인 Gmail 하루 100통 한도 내)

---

## 📁 구성

```
gas/
 ├─ Code.gs            # 메인 로직 (발송·미리보기·자동발송·정기항목 등)
 ├─ Setup.gs           # 시트 생성 / 양식 통합 / 입력 형식
 └─ appsscript.json    # 권한(스코프) 매니페스트
docs/
 └─ 거래명세서 양식.xlsx  # PDF로 출력될 양식 (반드시 가져오기) — 예시 데이터로 채워져 있음
README.md
```

> 거래처/금액 등 모든 데이터는 **공개 저장소가 아니라 본인 구글 시트**에만 들어갑니다. 코드의 값들은 전부 **예시(placeholder)** 입니다.

## 🚀 설치

### 1) 양식 가져오기
1. 새 구글 시트 생성 (예: [sheets.new](https://sheets.new))
2. **파일 → 가져오기 → 업로드** → `docs/거래명세서 양식.xlsx` 선택
3. 가져오기 위치 **"새 시트 삽입"** 선택 → 가져오기
4. → `자료입력` / `세금계산서양식` 두 탭이 생기면 성공

### 2) 코드 붙여넣기
1. **확장 프로그램 → Apps Script**
2. ⚙ **프로젝트 설정 → "에디터에 `appsscript.json` 매니페스트 파일 표시" 체크**
3. 파일 3개를 맞춥니다:
   - `appsscript.json` ← `gas/appsscript.json` 내용
   - `Code.gs` ← `gas/Code.gs` 내용
   - `Setup.gs` ← (`+` → 스크립트, 이름 `Setup`) `gas/Setup.gs` 내용
4. 💾 저장

### 3) 설치 실행
1. 구글 시트로 돌아와 새로고침(F5) → 상단에 **`📨 거래명세서`** 메뉴 생성
2. **`📨 거래명세서 → ⚙ 관리 → 설치 / 갱신`** 클릭
3. 최초 1회 **권한 승인**: `고급 → (안전하지 않음으로) 이동 → 허용`
   - 구글 미인증 개인 스크립트라 뜨는 정상 단계입니다. 본인 시트/메일/드라이브만 사용합니다.
4. → `설정 · 거래처 · 정기항목 · 거래내역 · 세금계산서양식 · 발송로그` 시트 생성 + 양식이 한 시트로 통합됨

## ✏️ 무엇을 바꿔야 하나 (★ 여기만 채우면 됩니다)

### ① `설정` 시트 — 내 회사/발송 정보
| 키 | 넣을 값 | 비고 |
|----|---------|------|
| `공급자_상호` | 내 회사명 | PDF·메일에 표시 |
| `공급자_사업자번호` | 내 사업자번호 | **하이픈 있게/없게 아무거나** (자동 정렬). 사업자번호·고유번호(10자리) |
| `공급자_대표자` | 대표자명 | |
| `공급자_주소` | 사업장 주소 | |
| `공급자_업태` / `공급자_종목` | 업태 / 종목 | |
| `공급자_입금계좌` | 입금 계좌 | **메일 본문**에 들어감 |
| `관리자_이메일` | 발송 결과 요약 받을 내 메일 | 비어 있으면 요약 메일 안 옴 |
| `테스트_수신메일` | 테스트 받을 내 메일 | POC/테스트는 전부 이 주소로만 발송(실고객 X) |
| `발송일` | 자동발송 날짜 (1~31) | 드롭다운 |
| `발송시각` | 자동발송 시각 (0~23, 24시간제) | 드롭다운 |
| `청구월_기준` | `당월` / `전월` | 보통 당월 |
| `발송_활성화` | `TRUE`=실제 발송 / `FALSE`=PDF만 생성(보류) | 처음엔 FALSE로 테스트 권장 |

### ② `거래처` 시트 — 고객 명단
- **필수**: `거래처ID`(C001, C002 …), `상호`, `이메일`
- 선택: 사업자번호, 대표자, 주소, 업태, 종목 / `발송여부`(FALSE면 제외)
- 예시 3곳·테스터 1곳이 들어 있으니 **본인 거래처로 교체**하세요.

### ③ `정기항목` 시트 — 매달 청구할 내용 (주력 시트)
- `거래처ID`(목록 선택 → 옆에 상호 자동표시), `품목`, `수량`, `단가`, `사용여부`(TRUE)
- 여기 적어두면 매달 자동으로 거래명세서가 만들어집니다.

### ④ 양식의 **은행계좌** (PDF에 찍히는 계좌)
- `세금계산서양식` 시트 하단 **은행/계좌 칸**(`○○은행 000-00-000000 …`)은 양식에 직접 박혀 있습니다 → **본인 계좌로 수정**하세요.
- (메일 본문 계좌는 위 `설정`의 `공급자_입금계좌`, PDF 계좌는 이 양식 칸 — 둘 다 채우면 됩니다)

## 🗓 사용법 (매월)

1. **`정기항목`** 에 매달 보낼 내용 관리 (대부분 한 번 적어두면 끝)
2. **`📨 거래명세서 → 👀 미리보기`** → 발송 없이 PDF만 생성 → Drive에서 내용 확인
3. **`📧 이번 달 발송하기`** → 확인창에서 대상 건수 확인 후 발송
4. 또는 **`⚙ 관리 → 자동발송 켜기`** → 매월 `발송일`·`발송시각`에 자동 발송

### 발송 대상 규칙
**이번 달(청구월=당월) + 발송상태가 `미발송` + 거래처 `발송여부=TRUE`** 인 건만, **`발송_활성화=TRUE`** 일 때 발송됩니다.
- 특정 행을 안 보내려면 → `거래내역`의 그 행 **`발송상태`를 `보류`** 로 (다시 만들어지지도 않음)
- 한 번 보낸 건은 `발송완료`로 표시 → 중복 발송 안 됨

## ⚠️ 알아둘 점

- **개인 Gmail은 하루 100통** 한도 (Google Workspace는 1,500통). 소량이면 무관.
- **발송_활성화 = FALSE** 면 PDF만 만들고 메일은 안 갑니다 → 처음 테스트할 때 안전.
- **자동발송**은 구글 트리거 특성상 지정 시각의 **1시간 이내**에 실행됩니다(예: 15시 → 15:00~16:00). 즉시 확인은 `테스트: 1분 후 자동발송`을 쓰세요.
- **사업자번호** 칸은 사업자번호 또는 단체 고유번호(둘 다 10자리)용입니다. 전화번호 등은 넣지 마세요.
